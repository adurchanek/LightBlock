
- MULTI-ASSET RENAMER:

Easily find and replace parts of asset names. 

Open up the editor window from the menu bar: Window -> Mirza Beig -> Multi-Asset Renamer. 

Select the assets in the PROJECT VIEW you want to search. 
Edit the find and replace terms and click the button to begin.
